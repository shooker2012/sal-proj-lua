*sal-proj-lua-commands*

------------------------------------------------------------------------------
Show svn log.                                                         *SVNLog*

:SVNLog [path]

:SVNUpdate [path]

DEFAULT path is CURRENT FILE("%").

:SVNDiff [path] 

DEFAULT path is CURRENT FILE("%").

------------------------------------------------------------------------------

:SVNBlame [path]

DEFAULT path is CURRENT FILE("%").

:SVNCommit [path]

:SVNRevert [path]

DEFAULT path is CURRENT FOLDER(".").
